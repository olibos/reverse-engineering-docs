⚙️ La Méthode

01 // PASTE

Collez le code tel quel.
Pas besoin de nettoyage.
L'IA s'en occupe.

02 // ASK

Posez des questions ciblées :
"que fait ce code ?"
"si cette variable vaut ... ?"
"c'est quoi ce buffer ?"

03 // REWRITE

Demandez une réécriture propre avec les API natives — sans librairies externes.


💡 C'est comme faire du pair programming avec quelqu'un qui a lu toutes les RFC, specs et pages MDN qui existent.
