Déchiffrement de la clé AES depuis une chaîne binaire

MOI
Explique ce code : 
  
r.decryptAsmCrypto = function(e, t) {
  return a(e, function(e, r) {
    var n = function(e) {
      var t, r = e.length,
        n = new Uint8Array(r);
      for (t = 0; t < r; t++)
        n[t] = e.charCodeAt(t);
      return n;
    }(r.toString("binary"));

    return t.AES_CBC.decrypt(e, n).subarray(16);
  });
};

IA
Cette fonction déchiffre des données en AES-CBC.
La méthode interne convertit une chaîne binaire en Uint8Array.
Le .subarray(16) supprime l'IV ajouté en tête après déchiffrement.
